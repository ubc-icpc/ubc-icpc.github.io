General I/O?

Please use standard in and standard out!

Input --
Java: Scanner in = new Scanner(new BufferedReader(new InputStreamReader(System.in)));

C++: int x; cin >> x;

Output --
Java: System.out.println("asdf");
C: printf("%d %d", integer1, integer2);
C++: int x = 5; cout << x << endl;

E - Palindromic Pokemon Do things like 05:15 count as palindromic?

No, a palindrome reads the same forward and backwards. E.g. 11:11 is a palindrome, but 05:15 is not.
C - Lt. Surge's New Puzzle What does Incorrect Output mean for C?

It means your solution either: (a) took too many operations, (b) gave the wrong matching of wires to switches, or (c) gave a wire that was out of bounds.
C - Lt. Surge's New Puzzle Can I flip 2 or more switches at once? If so, how?

You can only flip 1 switch with each operation. To flip 2 switches, you need to send 2 separate FLIP operations.
